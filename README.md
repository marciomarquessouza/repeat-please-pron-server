## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## ⛏️ Built Using <a name = "built_using"></a>

- [Nest](https://docs.nestjs.com/) - Nest (NestJS) is a framework for building efficient, scalable Node.js server-side applications.
- [TypeORM](https://typeorm.io/) - TypeORM is an ORM (Object-relational mapping) that can run in NodeJS.
